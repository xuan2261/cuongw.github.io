## Features

* Ordering drinks, foods by table.
* Checkout, preview & print invoice.
* Profile feature.
* Manage drink, food & category.
* Manage table.
* Manage account.
* Sales report.

## Install

Clone this project:
```
$ git clone https://github.com/cuongw/ting-coffee.git
```
cd `ting-coffee`:
```
$ cd ting-coffee
```
Install packages for `flutter` apps:
```
$ flutter packages get
```
