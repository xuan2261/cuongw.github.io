## Features

* Ordering foods by table.
* Checkout, preview & print invoice.
* Profile feature.
* Manage food & category.
* Manage table.
* Manage account.
* Sales report.

## Install

Clone this project:
```
$ git clone https://github.com/cuongw/store-pattern.git
```
cd `store-pattern`:
```
$ cd store-pattern
```
Install packages for `flutter` apps:
```
$ flutter packages get
```
